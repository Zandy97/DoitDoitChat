## 프로젝트 선정이유 
평소 카카오톡을 사용하면서 채팅에 관심이 있었습니다. 그리고 평소에 개발 하던 방식이 아닌 
1:N 방식의 개발을 해보고 싶어서 채팅을 주제로 하였습니다.
